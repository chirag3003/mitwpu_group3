## Custom Charts & Graphics

### CAShapeLayer for Donut Charts

Used in: `CircularProgressView`, `SemicircularProgressView` (Home screen progress indicators)

| Resource | Description |
|----------|-------------|
| [Creating Custom Charts in iOS Using Swift](https://medium.com/@ipak.tulane/creating-custom-charts-in-ios-using-swift-959551e1f7eb) | Tutorial on building custom chart views using CAShapeLayer |
| [Apple CAShapeLayer Documentation](https://developer.apple.com/documentation/QuartzCore/CAShapeLayer) | Official documentation for CAShapeLayer class |

**What we learned:**
- Drawing circular paths using `UIBezierPath`
- Animating `strokeEnd` property for progress animations
- Configuring `lineCap`, `lineWidth`, and `strokeColor`
