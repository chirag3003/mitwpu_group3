# View Controllers

Organized by feature in the `Controller/` directory.

## Home

| Controller | Description |
|------------|-------------|
| `HomeViewController` | Main dashboard with health summaries |

## Meals

**Directory**: `Controller/Meals/`

| Controller | Description |
|------------|-------------|
| `MealViewController` | Main meal tracking screen with calendar |
| `MealDetailViewController` | Individual meal details |
| `MealDataViewController` | Meal data management |
| `AddMealModalViewController` | Add new meal modal |

## Symptoms

**Directory**: `Controller/Symptoms/`

| Controller | Description |
|------------|-------------|
| `SymptomViewController` | Symptom list and management |
| `AddSymptomViewController` | Add new symptom form |

## Profile

**Directory**: `Controller/Profile/`

| Controller | Description |
|------------|-------------|
| `ProfileTableViewController` | User profile display |
| `EditProfileViewController` | Edit profile form |
| `AllergyViewController` | Allergy list |
| `AddAllergyTableViewController` | Add allergy form |

## Documents

**Directory**: `Controller/Document/`

| Controller | Description |
|------------|-------------|
| `DocumentsViewController` | Document list |
| `DocumentUploadViewController` | Upload new document |
| `HealthReportViewController` | View health reports |
| `GenerateHealthViewController` | Generate health summary |
| `PrescriptionPageViewController` | Prescription viewer |
| `CustomCameraViewController` | Document camera capture |
